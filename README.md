# MAGD-Github-Project
A simple repository containing class assignments for media arts &amp; game design.

## Lab #1
> This lab is an introduction to P5JS. In this assignment we learned how to create a basic canvas with primitive shapes.

## Lab #2 
> This lab focuses on introducing choosing a color mode to the sketch with the ability to set colors based on RGB values.

## Lab #3 
> This lab introduces mouse position tracking alongside basic mathmatical functions P5JS offers, such as `pow()`, `abs()`, `sqrt()`, `dist()`, and `radians()`
